# MEMORY.md 提炼规则

`MEMORY.md` 是长期记忆层。
它不是周报、索引页、导航页，也不是详细复盘文件目录。

## 允许写入
只允许写：
- 长期规则
- 长期结论

## 禁止写入
绝对不要写：
- 下周动作
- 文件路径
- 索引
- 去哪里看详细文件
- 周报式摘要
- 临时现象
- 一次性问题
- 详细过程

## 写入前的基础前提
一条内容只有在同时满足以下两条时，才有资格继续评估是否写入 `MEMORY.md`：
1. **不是一次性细节**
   - 不是某次偶发情况
   - 不是某周特殊背景下才成立的局部信息
2. **能脱离原上下文独立成立**
   - 未来再次看到这条内容时，不依赖整篇周复盘也能理解并使用

## 正式阈值
在满足基础前提后，还必须至少满足以下三项中的两项：
1. **重复出现**
   - 同一周内多次出现
   - 或跨天 / 跨任务反复出现
   - 或本周再次出现过去复盘里提到过的同类规律 / 同类问题
2. **影响结果**
   - 明显影响结果质量、效率、返工、判断或用户体验
3. **未来还会遇到**
   - 属于高频场景、固定流程或系统性问题，未来仍有复用价值

## 未达阈值时怎么处理
如果候选内容未达阈值：
- 不更新 `MEMORY.md`
- 保留在周复盘详细文件中
- 可以写成“候选长期规则 / 候选长期结论”继续观察
- 但不能强行写入 `MEMORY.md`

## 输出规则
当更新 `MEMORY.md` 时：
- 只写提炼后的长期规则 / 长期结论
- 不要复制周复盘整段文字进去
- 不要把详细周复盘内容搬进长期记忆层

## 通知提醒
每次周复盘完成后，都要通知用户：
- 周复盘是否完成
- `MEMORY.md` 是否更新
- 如果更新：概述新增的长期规则 / 长期结论
- 如果未更新：明确说明本周无新增长期规则 / 长期结论
